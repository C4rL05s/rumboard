# **RUMBoard**

This repository is the frontend code for the website RUMBoard. Made as a project for a class with the intent of connecting students & associations across college campuses. 
To see the backend please visit https://github.com/C4rL05s/rumboard-backend

## **Frameworks utilized**

- Vuejs
- Express
- Tailwindcss
- Node.js

## **Project Setup**

```sh
cd ~/Desktop
npm create vue@latest
```
Once *npm create vue@latest* is run, answer the questions. Select to use **Router, Linter and Prettier** 

**NO EXPERIMENTAL FEATURES**

```sh
cd ~/Desktop/project-folder
npm install
npm run format
mkdir src/views
mkdir src/components
mkdir src/assets
npm run dev
```
